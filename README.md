## 🏗️ Featured Projects

### 🏢 [PropNest](https://github.com/sumayyashahul-git/PropNest) — Real Estate Microservices Platform
> Enterprise-grade property management system built on a microservices architecture

- **8 independent services** — Property, Booking, User, Notification, Document, Analytics, and more
- **Event-driven** communication via Azure Service Bus; fully containerized on Docker/AKS
- **Multi-frontend** — Angular 19 admin portal + React/Vite client portal with shared JWT auth
- **AI integration** — Azure Cognitive Services for document extraction from contracts and title deeds
- **Python scripts** — Pandas-based data pipelines for property analytics and reporting (FastAPI)
- *Stack: .NET Core 9 · Angular 19 · React · Azure · Docker · SQL Server · EF Core*
